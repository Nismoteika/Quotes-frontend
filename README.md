# Quotes frontend

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Deploy
1. `yarn build`
2. send to server dist/* -> /public/
3. send to server views -> /resources/
